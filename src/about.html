Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais do desenvolvimento frontend moderno, indo além da simples estilização de telas.

A aplicação consome a FakeStore API de forma assíncrona, trabalhando com estados de carregamento, manipulação de dados, organização de código e separação de responsabilidades. Todo o desenvolvimento foi feito utilizando JavaScript puro, reforçando a compreensão da base da linguagem sem abstrações de frameworks.

O foco do projeto está na construção da interface, experiência do usuário, estruturação do código e boas práticas, incluindo:

  • Consumo de APIs REST
  • Tratamento de estados (loading, erro e sucesso)
  • Organização e reutilização de código
  • Responsividade e UI consistente
  • Uso consciente de CSS e Tailwind via CDN

Alguns dos fundamentos aplicados neste projeto também são utilizados em sala de aula como apoio no ensino de frontend.
